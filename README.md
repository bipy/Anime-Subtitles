# Anime Subtitles (Simplified Chinese)
收集整理了大量的动漫字幕，主要语言为简体中文（SC），少量为繁体中文（TC）

全部字幕适配 BD 版资源，改名匹配文件结构并制作文件索引（见 Google 文档）

动漫相关信息来自 [Bangumi](https://bgm.tv)

## 如何单独下载一部分文件？
[GitZip](https://gitzip.org/)
